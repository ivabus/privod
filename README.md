# Privod (оптический привод)

Ejects CD/DVD/BD drive at random time (from 5 secs to 5 mins)

Linux only.

## Usage

```shell
cargo run <PATH_TO_DRIVE>
```

`PATH_TO_DRIVE` is usually `/dev/sr0`
